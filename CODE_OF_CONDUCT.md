# Code of Conduct

This project expects respectful, constructive participation.

## Expected Behavior

- Be kind and direct.
- Keep feedback focused on the work.
- Assume good intent while still naming problems clearly.
- Respect user privacy, especially around local files and images.

## Unacceptable Behavior

- Harassment or personal attacks
- Sharing private user data
- Posting credentials, tokens, or private images
- Derailing technical discussions with unrelated conflict

## Enforcement

Project maintainers may remove comments, close issues, or block participants who repeatedly ignore these expectations.
