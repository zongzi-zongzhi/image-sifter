# Image Sifter PRD

## 项目背景

用户在整理大量本地图片时，需要快速判断图片是否保留。手动打开图片、关闭、删除文件的流程重复且慢，也容易误删。

## 项目目标

创建一个 Windows 本地桌面小工具：用户选择一个图片文件夹后，可以逐张查看图片，通过键盘快速切换，并用 Enter 将当前图片送到系统回收站。

## 目标用户

需要快速清理本地图片文件夹的个人用户。

## 用户场景

用户打开程序，选择包含图片的文件夹。程序显示第一张图片。用户按左方向键查看上一张，按右方向键查看下一张，按 Enter 把当前图片送到回收站并自动进入下一张。

## 输入内容

- 用户选择的本地文件夹
- 文件夹内的图片文件
- 用户的键盘操作和按钮点击

## 输出结果

- 当前图片预览
- 当前浏览进度
- 被删除图片进入系统回收站
- 文件夹看完后的完成提示

## 核心功能

- 选择本地文件夹
- 读取常见图片格式
- 单图大预览
- 上一张和下一张切换
- Enter 快捷键删除到回收站
- 删除后自动跳到下一张
- 首次 Enter 删除前提示一次，降低误触风险

## 用户使用流程

1. 启动程序。
2. 点击“选择文件夹”。
3. 程序加载图片并显示第一张。
4. 使用左方向键或按钮查看上一张。
5. 使用右方向键或按钮查看下一张。
6. 使用 Enter 或按钮将当前图片送到回收站。
7. 当前文件夹没有图片可看时，程序显示完成状态。

## MVP 范围

- Windows 本地桌面程序
- 单文件夹、单层图片读取
- 支持 `.jpg`、`.jpeg`、`.png`、`.gif`、`.webp`、`.bmp`、`.avif`
- 图片删除使用系统回收站，不做永久删除
- 默认快捷键固定为：左方向键上一张，右方向键下一张，Enter 删除

## 非目标范围

- 图片编辑
- AI 识图或分类
- 云端同步
- 数据库管理
- 递归扫描子文件夹
- 自定义快捷键配置界面

## 后续扩展方向

- 支持扫描子文件夹
- 支持自定义删除快捷键
- 支持撤销上一次删除
- 支持缩略图列表
- 支持打包安装包

## 成功标准

- 用户可以启动程序并选择图片文件夹。
- 程序能逐张显示图片。
- 左右方向键可以切换图片。
- Enter 可以把当前图片送到回收站并跳到下一张。
- 没有图片或浏览结束时，界面给出清楚提示。
