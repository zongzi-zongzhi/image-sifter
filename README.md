# Image Sifter

Image Sifter 是一个本地图片清理桌面小工具：选择图片文件夹后，逐张查看图片，用快捷键快速保留或送到回收站。

## 解决的问题

整理大量图片时，手动打开、关闭、删除文件很慢。Image Sifter 把流程简化成一个专注窗口：看图、上一张、下一张、Enter 删除。

## 核心功能

- 选择本地图片文件夹
- 单图大预览
- 左方向键查看上一张
- 右方向键查看下一张
- Enter 将当前图片送到系统回收站
- 删除后自动跳到下一张
- 显示当前进度和文件名

## 当前 Demo 能力

当前版本是本地 MVP，支持 Windows 上运行 Electron 桌面应用。

支持图片格式：

- `.jpg`
- `.jpeg`
- `.png`
- `.gif`
- `.webp`
- `.bmp`
- `.avif`

## 安装方式

```bash
npm install
```

## 运行方式

```bash
npm start
```

Windows 上也可以直接双击：

```text
启动 Image Sifter.bat
```

如果依赖缺失，这个启动文件会自动运行 `npm install`。如果启动失败，窗口会停住并显示错误信息。

## 测试方式

```bash
npm test
```

## 使用示例

1. 启动程序。
2. 点击“选择文件夹”。
3. 使用 `←` 和 `→` 浏览图片。
4. 使用 `Enter` 将当前图片送到回收站。

首次按 Enter 删除时，程序会显示一次确认提示。

## 安全与隐私提示

- 程序只读取用户选择的本地文件夹。
- 程序不上传图片。
- 删除操作使用系统回收站，不做永久删除。
- 不要把个人图片提交到公开仓库。

## 已知限制

- 当前只扫描所选文件夹的第一层图片，不递归扫描子文件夹。
- 当前快捷键固定为：左方向键、右方向键、Enter。
- 当前不提供撤销删除功能，但可以从系统回收站手动还原。
